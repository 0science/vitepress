# 入门

## 历史

C 语言是一种通用的高级语言，最初是由丹尼斯·里奇在贝尔实验室为开发 UNIX 操作系统而设计的。C 语言最开始是于 1972 年在 DEC PDP-11 计算机上被首次实现。

## 特点

- 设计特性
- 高效性
- 可移植性
- 强大灵活
- 面相过程

## 应用范围

- 操作系统
- 嵌入式系统
- 应用软件
- 游戏开发

## 语言标准

- C89：奠定了现代C语言基础
- C99：最重要的功能扩展
- C11：稳定性和安全性增强

## 编译与执行过程


- 预处理
- 编译
- 汇编
- 链接

## 第一个C程序
输出Hello World!

```c
#include <stdio.h>
int main(){
    printf("Hello World!");
    return 0;
}
```

## 总结

- C语言既不是汇编语言，也不是高级语言，而是一种通用的中级语言。
- C语言编译过程就是源代码经过预处理、编译、汇编和链接四个阶段，最终转换为可执行文件。
- 可执行文件是"完整独立的产品"，而源文件和目标文件是"生产过程中的中间产物"。
