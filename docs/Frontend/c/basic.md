# 基础知识

## printf函数

`"%[标志][宽度][.精度][长度]类型"`

```
printf("%d",100);
printf("%d%d%d",100,200,300);
printf("%5d",100);
printf("%-5d",100);
printf("%05d",100);
printf("%f",2.1);
printf("%.2f",3.1415926);
printf("%c",'h');
printf("%s","hello world");

```

|格式符|含义|
|--|--|
|s| 字符串|
|c| 字符|
|d| 十进制整数|
|u| 十进制无符号整数|
|o| 八进制整数|
|x| 十六进制无符号整数|
|X| 十六进制无符号整数|
|f| 浮点数|
|e| 科学计数法|
|E| 科学计数法|
|p| 指针|

|标志符|含义|
|--|--|
|-| 左对齐|
|+| 显示正负号|
|0| 数字前面补0|
|空格| 显示正数时，前面补空格，负数时，前面补-|
|#| 显示八进制时，前缀加0<br>显示十六进制数时，前缀加0x或0X|

## scanf函数

```
int a;
float b;
char c;
scanf("%d",&a);
scanf("%f",&b);
scanf("%c",&c);

```

|字符|含义|

|\t| 制表符|
|\n| 换行符|
|\v| 垂直换行符|
|\f| 换页符|
|\r| 回车符|

## 关键字

C语言中具有特殊含义和固定功能的预定义标识符。
关键字必须小写。

## 标识符
程序中用来命名变量、函数、数组、结构体等程序实体的自定义名称。

+ 只能由字母、数字、下划线组成
+ 不能包含除下划线以外的其它特殊字符串
+ 不能以数字开头
+ 不能是C语言中的关键字
+ 严格区分大小写
